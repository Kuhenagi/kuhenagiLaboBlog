 1.GitHubにサインイン
 割愛したいぐらい簡単です。
 まずここに行ってください。



GitHubに登録するを押しましょう。


上から使いたいハンドルネーム、メールアドレス、
GitHubアカウントのパスワード(作成後のログインに使います。)
 すべて埋めたらロボット化の検査があるのでかいくぐりましょう。
 そしてメッセージが届くのでメールを見て作成終了です。
 2.レポジトリ作成


こんな画面になっていると思うので＋をクリックして、
 New repositoryを押してください。


Repository nameには作りたいページの名前を入れましょう。(今回はnetshopにします。)
 Descriptにはこのネットショップの説明を入れます。(「kuhenagiの期間限定カードショップです。」にしました。)
 あとはAdd a readme fileにチェックして緑のやつを押しましょう。


この画面になれば勝利です。
